发信站: BBS 未名空间站 (Sat Sep 28 00:06:46 2013, 美东)

最近骑驴找马，面经一般般，但是要钱颇有一些心得。据说同样的职位印度人平均比中
国人高20%，这20%就是人家要来的。因此把经验和大家分享分享，赚几个包子。

如果你不讨价还价，HM搞不好认为你不qualify，又或者会很快跳槽。：P

首先大原则是要遵守游戏规矩。讨价还价是正常的，答应又反悔是最不好的，还不如先
去工作半年再说。有些国人拿到offer不敢讲价，生怕触怒对方，然后有了更好的offer
又想反悔之前的诺言，这最要不得，根本是吃了亏还要挨骂。

然后要知己知彼，求职者自己是一边，另一边是HR，HM，有时候还有第三方的猎头。猎
头的commission机制最近似乎有些改变，不再有动力帮我们讨要更高的薪水，而是更加
愿意低价成交，速度第一。有一点一定要注意，如果通过猎头找工作，一定不要跳过猎
头，否则公司也会认为你不懂规矩。HR会从HM那里要个薪水范围，然后尽量以最低的价
钱，最快的速度雇HM要的人。至于这个人是否合格，不是HR考虑的事情。所以无论如何
，多要个20%左右是可以的。如果自己感觉面试结果很好，可以要更多。这时HR就会去
询问HM，而对于HM来说，钱反正不是自己出，找到能干活的人最重要。所以大家尽管要。

如果一开始对方问你现在的薪水或者期待的薪水，就说市场平均，然后反问，这个职位
的薪水范围。尽量避免这个问题，实在不行，给个范围，不要给确定的数字。对了，玻
璃门上的薪水似乎都是偏低的，所以只能当成下限。


薪水谈判最好能有多个offer，这样可以leverage，但是这样做也是有技巧的。最重要
的仍然是一旦答应，就必须去。注意如果你说个数，然后人家match了，这样也默认你
应该去了。所以要说：我从另一家拿到了多少多少，而且这是negotiable的。这样就仍
然有余地。

谈判最好不要超过两轮，避免给人贪婪无信的印象。这里的关键就是拖，跟对方说，我
现在还在等待更多的信息，无法决定。如果对方威胁要收回offer，而你觉得后面的
offer更好，那也就表示理解好了。他们多半不会真的收回的，就算收回了，这公司可
能并不是很器重你，那不去也好。

如果你说了个数，人家答应了，但是你又反悔了，也还是有办法的。我自己这里就犯了
个错误，把A家offer的数字告诉B家了，其实告诉存在这个offer就好了。

然后B家比A家多出1万，这时他们就认为我会去了。但是我告诉他们，这个数字还是太
低，他们有些生气了。

这时我就告诉他们，我说的A家的数字没有包括20%的bonus，所以如果他们beat这个还
不够。如果他们出xxx这么多，我就happily accept。

这样base就定下来了，一般情况下，接着谈bonus，PTO，绿卡，等等。随着对方答应你
的更多，他们就更希望你去了，呵呵。

职场人心：谈判薪水的艺术

小时候和妈妈上菜市场，常看到妈妈和小贩讨价还价。有时争争吵吵，有时苦苦哀求，你来我往地大战几个回合，倒也不伤和气，最后大都是顺利成交，下次再战。那时候我太年轻，不懂得欣赏讨价还价的艺术，只觉得有点尴尬和不屑：要买就买，不买拉到，何苦不干不脆，拖泥带水的？

若干年后，我开始自己赚钱自己花，亲身体会“钱到用时方恨少”的滋味，巴不得有妈妈杀价的本领，一块钱可以当两块用。奈何讨价还价的事看来简单，真要实行却不知所措，鼓起勇气试一次，就带来“不买拉到”的尴尬。回头想想，当年我不但没有讨价还价的技巧，也缺乏正确的观念：讨价还价固然是在谈价钱，更是在谈关系！

关系怎么谈？如果新公司给你一个offer，年薪八万，要就收下，不要拉倒，双方有什么关系？反过来说，如果你还价，年薪十万，否则免谈，双方有什么关系？谈判薪水是不是你输我赢的敌对关系？

其实，双方利害非常一致。公司花了许多时间和人力，好不容易找到他们想要的人，很担心这个人不来，他们必须重新来过，费时费力。你也是好不容易找到你要的工作，深恐饭碗还没到手就打破了。然而，公司里对薪水有些限制，比方说你的薪水和老板比，和市场价格比，和其他同事比，都有准则；单位的开销也不能超过预算。谈薪水就是要在这些限制下，得到最有利的offer。这件事要靠你未来的老板替你奔走出力，所以谈薪水要避免敌对的心态，而要设法让老板站在你这边。

谈薪水有一个简单的套磁：很喜欢这个机会，公司好，工作好，同事好，（讲一堆好话），只是你在金钱上有点问题，希望老板帮忙解决这个问题。这种讲法把你和老板放在同一条阵线上，一同解决“问题”，谈得不欢而散的可能性很小。“金钱上的问题”随你讲，搬家需要买房子，要应付目前的房贷，要付子女的学费，要付以前的学生贷款，要帮助国内的家人；反正只是找借口要钱，大家心照不宣。谈这些“金钱上的问题”也可以拉近彼此的距离，因为每个人都觉得钱不够用，谈着谈着关系就谈出来了。

奈何形势比人强。客观形势上，有时候你占优势，几个机会等着你选；有时候你处劣势，深恐丢了唯一的机会。用“金钱上的问题”来谈薪水，是一种攀交情拉关系的谈法，无论优势劣势都可以用。只是，有些人比较重视势力，所以如果你占优势，不妨暗示一下：“我的朋友在一家国际银行做事，他老板几次要找我过去，实在对我很好。只是我对贵公司的工作更有兴趣，很希望能来这里上班”。这句话虽然暗示你的势力，可以选择其他的工作机会，但是听起来还像是在谈关系。通常雇方会问你那家银行的offer，决定要如何应付。如果你主动把其他公司offer的细节晒出来，一样一样要求跟进，你可能太强势，让老板觉得被逼迫，难免伤了关系。

谈判薪水的艺术其实也是人际关系的艺术。这个世界大家都说人人平等，却都知道实际上人人不平等。大家都怕被别人踩在脚底下，却不免争先往高处爬，把别人踩在脚底下。人际关系的艺术就在看似不平等的环境中，开拓平等的互动，建立“人同此心”的情感；同时灵巧地运用势力，表达自己的选择，也尊重别人的选择。在谈判薪水的过程中，未来的老板其实也在观察你人际关系的能力，决定要不要加码下注。

也许只有书生才会靠读书学习讨价还价的技巧。回想当年的菜市场，小贩和顾客讨价还价的对话，到处都是攀交情，拉关系，暗示势力，暗示选择的例子。人际关系是人的本能，不是什么大学问。几千年来，大家都是混一混就学会了。现代书生擅长考试，读篇报告算是临阵磨枪吧！

小弟不才，在一大公司做小经理。有关薪资的一些情况，可供参考。

在薪资谈判中，有两位重要人物－－Corporate Recruiter 和 Hiring Manager。

Corporate Recruiter

大公司一般有自己的 Corporate Recruiter。也就是所谓的“猎头”。Recruiter 负责在市场上找人（找简历），过滤后送给 Hiring Manager。面试后，如果 Hiring Manager 觉得满意，Recruiter 负责准备聘书（Offer Letter）。

Recruiter 通常要负责不同部门的招聘，不可能精通各种专业，因此也不可能判断你是大师级还是菜鸟级。如果你想凭专业水平和 Recruiter 谈薪资，那是对牛弹琴。

大公司对每个职位的薪资会作市场调查，然后制定一个范围。这个范围可以很广（譬如 79k - 119k）。这个范围的中值（譬如 99k）就是基准点。Recruiter 一般会以基准点为基础来和你谈判。

另外 Recruiter 也清楚公司的福利，如 401k match, Medial Insurance，Stock Option，ESPP，Bonus Plan 等。这些东西 Hiring Manager 反而不一定清楚。

Hiring Manager

你将来的顶头上司（如果你被聘请的话）。Hirging Manager 精通专业，会明白你的业务水平。Hiring Manager 会把你的水平告诉 Recruiter。Recruiter 就会大概知道给你什么薪资。如果你的水平是 Above Average，你的薪资就会是基准点以上（譬如 > 99k）。反之，如果 Hiring Mangaer 说你的业务水平是 Below Average，你的薪资就会是基准点以下（譬如 < 99k）。

要在谈判中取得主动，你要明白关键人物的处境。

作为 Recruiter，首要任务是完成招聘的任务。至于是否聘请到真正的人才，或者是否做得长久，都不是他的责任。（那是 Hiring Manager 的责任）。夸张地说，如果 Hiring Manager 愿意请一头猪，Recruiter 是不会反对的，只要快点完成招聘任务就成了。

因此，只要 Hiring Manager 觉得你还可以，Recruiter 是希望快点和你签合同的。市场上或许有比你更好的人选，但英语有句偈语叫“One bird in hand is better than two birds in the bushes”。Recruiter 才不想节外生枝。明白这一点，你就可以放心和 Recruiter 讨价还价。

作为 Hiring Manager，他是希望请到能干活的人。至于这个人是否 overpaid，他不关心（反正不是他的钱）。就拿我来说，当 Recuiter 问我“candidate 要求 $xxx，你觉得怎样？”我的回答总是“他工资比我高也没问题，只要是公司允许就行了。”

因此，你根本不需要和 Hiring Manager 讨价还价。只需尽力表现你的业务水平就行了。

好了，如果 Hiring Manager 那边过关了，怎样和 Recruiter 谈薪资呢？

首先，对方的 Offer 肯定是高于你目前的薪资的。如果低于的话，恐怕你也不太有兴趣。因此你只是想把利益最大化（尽量接近对方薪资范围的上限）。

你谈判的筹码通常有以下几种：

1. W2 或 Paystub

如果最近的 W2 或 Paystub 有“额外收入”（譬如 bonus, commission，overtime 等），那是一件利器。比方说，你一般的 Paystub 是 $3000，但最近有一张 Paystub 是 $5000 （因为有 bonus），你可以把这张 bonus 给对方看。当然，对方聪明的话，会留意到。但无论如何，他的印象中你的 paycheck 就是在 $5000 的水平，而不是 $3000 的水平。

2. Stock Option, ESPP（Employee Stock Purchase Plan），RSU（Restricted Stock Unit）

这些和股票有关的福利不是每家公司都有，收益也不固定。如果你目前有这些福利，而对方的 Offer 中没有，你的谈判空间就大了。

3. Medical Insurance

每家公司的 Medical Plan 都不一样。从 premium 到 copay 到 deductibe 都有差异。我有个朋友就以这个做文章，把基本工资提升了 10k。要知道老美数学都不太行，但又很迷信数字，因此被我那朋友忽悠了。

4. Living expense

各地的生活指数不一样，也可以用作谈判的筹码。在网上搜索 relocation, pay scale 等会找到一些有用的参考数据。

5. Personal/family concern

如果实在拿不出什么数据，就用个人/家庭为借口。譬如老婆明明是家庭主妇，你可以说她工作收入不错。如果你要转换工作，由于时间、地点的变化，她的收入就没有了。又或者说为了新工作，你要送 Baby 去 daycare，不小的开支，等等。反正发挥你的想象力。

6. Base salary vs. sign-on bonus

上面谈到，base salary 是有一个范围的。你想超出上限，那是神仙也没有办法。但许多人往往忽略了 sign-on bonus。

Recruiter 通常有一定的 sign-on bonus 权限，从几千到几万不等。Base salary 也许他无能为力，但他可以给你 sign-on bonus。但必须得有个理由（Business Justification）。这个也是发挥想象力了。常见的例子如：“之前的公司支付了你的培训费。你若离开，必须返还”，“Bonus 在季末发放，现在是季初。如果你现在离开，bonus 就泡汤了”......等等。
