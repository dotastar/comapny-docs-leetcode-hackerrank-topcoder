将自然数和其因子列出如下：
1：1
2：1，2
3：1，3
4：1，2，4
5：1，5
6：1，2，3，6
...
可以看到，因子个数为3的最小自然数是4，因子个数为4的最小自然数为6。
给定自然数N，求满足因子个数等于N的最小自然数。


dozerg:
给定一个自然数M，求其因子个数的算法如下：
将M分解成素数因子的乘积：
M = P1^p1 * P2^p2 * ... * Pn^pn
令C0 = 0，
C1 = C0 * (p1 + 1) + p1
C2 = C1 * (p2 + 1) + p2
...
Cn = Cn-1 * (pn + 1) + pn
则(1 + Cn)为最后的因子个数。

那么反过来，知道了Cn(= N - 1)，实际就是找到一个序列{p1, p2, ..., pn}满足上面的多项式。
由于这个序列不是唯一的，所以需要在多个序列里找到使M最小的一个。
假设这个序列是{p1, p2, ..., pn}，求M的过程为：
将{p1, p2, ..., pn}从大到小排序，则p1 >= p2 >= ... >= pn
令Pn表示从小到大的第n个素数，则：
M = P1^p1 * P2^p2 * ... * Pn^pn
